🚀 STEP 3: PUSH TO GITHUB
--------------------------

After creating the repository, GitHub will show you commands.
Run these commands in your terminal:

Option A - If you see HTTPS URL:
---------------------------------

git branch -M main
git push -u origin main


Option B - If you see SSH URL:
-------------------------------
git remote add origin git@github.com:YOUR_USERNAME/gov-e-services-portal.git
git branch -M main
git push -u origin main


📝 DETAILED STEPS:
------------------

1. Copy the commands from GitHub (replace YOUR_USERNAME with your actual username)

2. Open Command Prompt or PowerShell in your project folder:
   C:\xampp\htdocs\Project

3. Run the commands one by one:

   Command 1: Add remote repository

   Command 2: Rename branch to main
   git branch -M main

   Command 3: Push to GitHub
   git push -u origin main

4. Enter your GitHub credentials when prompted:
   - Username: Your GitHub username
   - Password: Your GitHub Personal Access Token (not your password!)


🔑 GITHUB AUTHENTICATION
-------------------------

GitHub no longer accepts passwords for Git operations.
You need to create a Personal Access Token:

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token" → "Generate new token (classic)"
3. Give it a name: "Gov E-Services Portal"
4. Select scopes: ☑ repo (all repo permissions)
5. Click "Generate token"
6. COPY THE TOKEN (you won't see it again!)
7. Use this token as your password when pushing
