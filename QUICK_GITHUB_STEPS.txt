🚀 UPLOAD TO GITHUB (3 STEPS):
───────────────────────────────

STEP 1: Create Repository on GitHub
────────────────────────────────────
1. Go to: https://github.com/new
2. Repository name: gov-e-services-portal
3. Description: Government e-services portal
4. Visibility: Public
5. ⚠️ DON'T check "Initialize with README"
6. Click "Create repository"


STEP 2: Get Your Personal Access Token
───────────────────────────────────────
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Name: "Gov Portal Upload"
4. Check: ☑ repo
5. Click "Generate token"
6. COPY THE TOKEN (save it somewhere!)


STEP 3: Push to GitHub
───────────────────────
Run these commands in PowerShell:

cd C:\xampp\htdocs\Project

git remote add origin https://github.com/YOUR_USERNAME/gov-e-services-portal.git

git branch -M main

git push -u origin main

When prompted:
- Username: Your GitHub username
- Password: Paste your Personal Access Token


✅ DONE! Your project is now on GitHub!

View at: https://github.com/YOUR_USERNAME/gov-e-services-portal


💡 EASIER METHOD: Use GitHub Desktop
─────────────────────────────────────
1. Download: https://desktop.github.com/
2. Sign in to GitHub
3. Add existing repository: C:\xampp\htdocs\Project
4. Click "Publish repository"
5. Done!
